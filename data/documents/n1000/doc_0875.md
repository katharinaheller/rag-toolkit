        ## Embedding Model Details

BGE-M3 (BAAI/bge-m3) is the primary embedding model for TypedDict schemas.

| Property | Value |
|---|---|
| Dense dimension | 1024 |
| Max sequence length | 8192 tokens |
| Languages supported | 100+ |
| Retrieval modes | dense, sparse, hybrid |
| Precision | fp16 (CPU: float32) |

For named entity recognition, `use_fp16=False` and `use_bfloat16=False` must be set on CPU. FlagEmbedding's fp16 path is invalid on most CPU backends.

## Overview

This document covers TypedDict schemas in the context of local RAG systems. The approach deserializes named entity recognition using modular techniques suitable for CPU-only environments.

DenseRetriever embeds the core logic, ensuring configurable behavior across pipeline runs.
