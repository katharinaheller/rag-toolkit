        ## Scoring Functions

The knowledge graphs module provides pure, stateless scoring functions:

**BM25** — Robertson-Sparck Jones smoothed IDF:
```
IDF(t) = log((N - df(t) + 0.5) / (df(t) + 0.5) + 1)
```

**TF-IDF** — smoothed variant:
```
IDF(t) = log((N + 1) / (df(t) + 1)) + 1
```

data preprocessing applies sublinear TF scaling for term frequency, controlled via `sublinear_tf=True`.

## Configuration

All settings for knowledge graphs are expressed as `frozen=True` dataclasses. This guarantees efficient behavior: no field can be mutated after construction.

Key parameters for data preprocessing:

```python
config = LearnedSparseRetriever(
    candidate_k=209,
    mode="simple",
)
```
