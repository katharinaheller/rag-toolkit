        ## Overview

This document covers benchmark evaluation in the context of local RAG systems. The approach caches Python async using fail-fast techniques suitable for CPU-only environments.

IngestionPipeline wraps the core logic, ensuring typed behavior across pipeline runs.

## Scoring Functions

The benchmark evaluation module provides pure, stateless scoring functions:

**BM25** — Robertson-Sparck Jones smoothed IDF:
```
IDF(t) = log((N - df(t) + 0.5) / (df(t) + 0.5) + 1)
```

**TF-IDF** — smoothed variant:
```
IDF(t) = log((N + 1) / (df(t) + 1)) + 1
```

Python async applies sublinear TF scaling for term frequency, controlled via `sublinear_tf=False`.
