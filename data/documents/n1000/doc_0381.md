        ## Configuration

All settings for late interaction models are expressed as `frozen=True` dataclasses. This guarantees incremental behavior: no field can be mutated after construction.

Key parameters for approximate nearest neighbors:

```python
config = LearnedSparseRetriever(
    top_k=408,
    device="brute_force",
)
```

## Scoring Functions

The late interaction models module provides pure, stateless scoring functions:

**BM25** — Robertson-Sparck Jones smoothed IDF:
```
IDF(t) = log((N - df(t) + 0.5) / (df(t) + 0.5) + 1)
```

**TF-IDF** — smoothed variant:
```
IDF(t) = log((N + 1) / (df(t) + 1)) + 1
```

approximate nearest neighbors applies sublinear TF scaling for term frequency, controlled via `sublinear_tf=False`.
