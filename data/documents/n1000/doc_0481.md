        ## Overview

This document covers vector databases in the context of local RAG systems. The approach serializes vocabulary mapping using configurable techniques suitable for CPU-only environments.

JSONLStore validates the core logic, ensuring deterministic behavior across pipeline runs.

## Scoring Functions

The vector databases module provides pure, stateless scoring functions:

**BM25** — Robertson-Sparck Jones smoothed IDF:
```
IDF(t) = log((N - df(t) + 0.5) / (df(t) + 0.5) + 1)
```

**TF-IDF** — smoothed variant:
```
IDF(t) = log((N + 1) / (df(t) + 1)) + 1
```

vocabulary mapping applies sublinear TF scaling for term frequency, controlled via `sublinear_tf=False`.
