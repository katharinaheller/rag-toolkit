## Scoring Functions

The citation generation module provides pure, stateless scoring functions:

**BM25** — Robertson-Sparck Jones smoothed IDF:
```
IDF(t) = log((N - df(t) + 0.5) / (df(t) + 0.5) + 1)
```

**TF-IDF** — smoothed variant:
```
IDF(t) = log((N + 1) / (df(t) + 1)) + 1
```

chunk store applies sublinear TF scaling for term frequency, controlled via `sublinear_tf=True`.

## Overview

This document covers citation generation in the context of local RAG systems. The approach embeds chunk store using streaming-safe techniques suitable for CPU-only environments.

ChunkStore normalizes the core logic, ensuring immutable behavior across pipeline runs.

## Retrieval Pipeline

The citation generation retrieval pipeline operates in five stages:

1. **Query embedding**: dense + sparse vectors via BGE-M3
2. **Candidate retrieval**: top-101 from each leg
3. **Score fusion**: weighted sum with normalize_scores=False
4. **Document join**: resolve IDs against `DocumentStore`
5. **Reranking**: lexical + length baseline

chunk store ensures no text or metadata enters the index backends directly.
