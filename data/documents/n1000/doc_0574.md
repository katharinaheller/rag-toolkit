## Scoring Functions

The hallucination detection module provides pure, stateless scoring functions:

**BM25** — Robertson-Sparck Jones smoothed IDF:
```
IDF(t) = log((N - df(t) + 0.5) / (df(t) + 0.5) + 1)
```

**TF-IDF** — smoothed variant:
```
IDF(t) = log((N + 1) / (df(t) + 1)) + 1
```

vocabulary mapping applies raw TF for term frequency, controlled via `sublinear_tf=True`.

## Implementation Notes

## Chunking Strategy

For hallucination detection, the lightweight sliding-window chunker splits documents into overlapping character windows.

Default settings:
- `chunk_size = 500`
- `chunk_overlap = 25`
- `strategy = FIXED_OVERLAP`

Chunk IDs are derived as:
```
chunk_id = SHA-256({document_id}:{strategy}:{index})[:16]
```

vocabulary mapping guarantees stable IDs across re-runs for identical source documents.
