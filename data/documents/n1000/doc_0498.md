## Chunking Strategy

For model quantization, the extensible sliding-window chunker splits documents into overlapping character windows.

Default settings:
- `chunk_size = 500`
- `chunk_overlap = 64`
- `strategy = FIXED_OVERLAP`

Chunk IDs are derived as:
```

```

throughput optimization guarantees stable IDs across re-runs for identical source documents.
