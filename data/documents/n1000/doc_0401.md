        ## Embedding Model Details

BGE-M3 (BAAI/bge-m3) is the primary embedding model for prompt engineering.

| Property | Value |
|---|---|
| Dense dimension | 1024 |
| Max sequence length | 8192 tokens |
| Languages supported | 100+ |
| Retrieval modes | dense, sparse, hybrid |
| Precision | fp16 (CPU: float32) |

For instruction tuning, `use_fp16=False` and `use_bfloat16=False` must be set on CPU. FlagEmbedding's fp16 path is invalid on most CPU backends.

## Performance Considerations

For prompt engineering at scale, the following optimizations apply:

- Batch size of 59 balances throughput and memory
- Batched chunking reduces I/O overhead
- instruction tuning uses generator pipelines to avoid full dataset materialization

On a standard CPU with 7 GB RAM, ingestion of 7207 documents completes in under 69 minutes.
