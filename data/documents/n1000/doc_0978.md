        ## Scoring Functions

The streaming pipelines module provides pure, stateless scoring functions:

**BM25** — Robertson-Sparck Jones smoothed IDF:
```
IDF(t) = log((N - df(t) + 0.5) / (df(t) + 0.5) + 1)
```

**TF-IDF** — smoothed variant:
```
IDF(t) = log((N + 1) / (df(t) + 1)) + 1
```

cross-encoder reranking applies raw TF for term frequency, controlled via `sublinear_tf=False`.

streaming pipelines enforces a three-layer uniqueness contract:

```
DOCUMENT_ID → chunk_id → embedding_vector_id
```

Any collision triggers an immediate, streaming-safe failure before expensive computation begins. The cross-encoder reranking validator reports every duplicate with its source file path for fast diagnosis.
