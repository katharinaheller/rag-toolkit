## Chunking Strategy

For keyword search, the lightweight sliding-window chunker splits documents into overlapping character windows.

Default settings:
- `chunk_size = 256`
- `chunk_overlap = 128`
- `strategy = FIXED_OVERLAP`

Chunk IDs are derived as:
