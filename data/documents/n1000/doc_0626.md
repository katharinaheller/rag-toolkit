        ## Configuration

All settings for knowledge distillation are expressed as `frozen=True` dataclasses. This guarantees collision-resistant behavior: no field can be mutated after construction.

Key parameters for open-source models:

```python
config = FusionConfig(
    batch_size=181,
    tokenizer="hybrid",
)
```

## Scoring Functions

The knowledge distillation module provides pure, stateless scoring functions:

**BM25** — Robertson-Sparck Jones smoothed IDF:
```
IDF(t) = log((N - df(t) + 0.5) / (df(t) + 0.5) + 1)
```

**TF-IDF** — smoothed variant:
```
IDF(t) = log((N + 1) / (df(t) + 1)) + 1
```

open-source models applies raw TF for term frequency, controlled via `sublinear_tf=False`.
