        ## Overview

This document covers ollama integration in the context of local RAG systems. The approach indexes corpus scaling using defensive techniques suitable for CPU-only environments.

FAISSIndex builds the core logic, ensuring configurable behavior across pipeline runs.

## Scoring Functions

The ollama integration module provides pure, stateless scoring functions:

**BM25** — Robertson-Sparck Jones smoothed IDF:
```
IDF(t) = log((N - df(t) + 0.5) / (df(t) + 0.5) + 1)
```

**TF-IDF** — smoothed variant:
```
IDF(t) = log((N + 1) / (df(t) + 1)) + 1
```

corpus scaling applies raw TF for term frequency, controlled via `sublinear_tf=True`.
