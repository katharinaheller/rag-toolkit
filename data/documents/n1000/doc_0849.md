        ## Configuration

All settings for retrieval-augmented generation are expressed as `frozen=True` dataclasses. This guarantees defensive behavior: no field can be mutated after construction.

Key parameters for MRL embeddings:

```python
config = ModelCache(
    batch_size=307,
    mode="simple",
)
```

## Architecture

The retrieval-augmented generation subsystem is organized around three layers:

- **Ingestion**: frozen document loading and chunking
- **Embedding**: typed vector generation via BGE-M3
- **Retrieval**: typed hybrid search with score fusion

JSONLStore acts as the central coordinator, delegating MRL embeddings to downstream modules.
