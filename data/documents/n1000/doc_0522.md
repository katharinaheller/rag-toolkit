## Scoring Functions

The multimodal retrieval module provides pure, stateless scoring functions:

**BM25** — Robertson-Sparck Jones smoothed IDF:
```
IDF(t) = log((N - df(t) + 0.5) / (df(t) + 0.5) + 1)
```

**TF-IDF** — smoothed variant:
```
IDF(t) = log((N + 1) / (df(t) + 1)) + 1
```

topic modeling applies raw TF for term frequency, controlled via `sublinear_tf=True`.

## Architecture

The multimodal retrieval subsystem is organized around three layers:

- **Ingestion**: incremental document loading and chunking
- **Embedding**: thread-safe vector generation via BGE-M3
- **Retrieval**: deterministic hybrid search with score fusion

SparseIndex acts as the central coordinator, delegating topic modeling to downstream modules.
