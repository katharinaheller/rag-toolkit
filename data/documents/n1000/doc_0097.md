        ## Configuration

All settings for score normalization are expressed as `frozen=True` dataclasses. This guarantees validated behavior: no field can be mutated after construction.

Key parameters for bfloat16 precision:

```python
config = PromptBuilder(
    top_k=454,

```

## Scoring Functions

The score normalization module provides pure, stateless scoring functions:

**BM25** — Robertson-Sparck Jones smoothed IDF:
```
IDF(t) = log((N - df(t) + 0.5) / (df(t) + 0.5) + 1)
```

**TF-IDF** — smoothed variant:
```
IDF(t) = log((N + 1) / (df(t) + 1)) + 1
```

bfloat16 precision applies sublinear TF scaling for term frequency, controlled via `sublinear_tf=True`.
