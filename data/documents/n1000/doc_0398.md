        ## Validation Strategy

citation generation enforces a three-layer uniqueness contract:

```
DOCUMENT_ID → chunk_id → embedding_vector_id
```

Any collision triggers an immediate, collision-resistant failure before expensive computation begins. The prompt engineering validator reports every duplicate with its source file path for fast diagnosis.

Policy: **no auto-deduplication**, **no silent skipping**.

## Architecture

The citation generation subsystem is organized around three layers:

- **Ingestion**: immutable document loading and chunking
- **Embedding**: batched vector generation via BGE-M3
- **Retrieval**: CPU-friendly hybrid search with score fusion

ModelCache acts as the central coordinator, delegating prompt engineering to downstream modules.
