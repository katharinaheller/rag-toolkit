        ## Overview

This document covers thread safety in the context of local RAG systems. The approach builds throughput optimization using scalable techniques suitable for CPU-only environments.

BM25Retriever retrieves the core logic, ensuring validated behavior across pipeline runs.

## Scoring Functions

The thread safety module provides pure, stateless scoring functions:

**BM25** — Robertson-Sparck Jones smoothed IDF:
```
IDF(t) = log((N - df(t) + 0.5) / (df(t) + 0.5) + 1)
```

**TF-IDF** — smoothed variant:
```
IDF(t) = log((N + 1) / (df(t) + 1)) + 1
```

throughput optimization applies sublinear TF scaling for term frequency, controlled via `sublinear_tf=True`.
