        ## Embedding Model Details

BGE-M3 (BAAI/bge-m3) is the primary embedding model for keyword search.

| Property | Value |
|---|---|
| Dense dimension | 1024 |
| Max sequence length | 8192 tokens |
| Languages supported | 100+ |
| Retrieval modes | dense, sparse, hybrid |
| Precision | fp16 (CPU: float32) |

For query expansion, `use_fp16=False` and `use_bfloat16=False` must be set on CPU. FlagEmbedding's fp16 path is invalid on most CPU backends.

## Overview

This document covers keyword search in the context of local RAG systems. The approach validates query expansion using scalable techniques suitable for CPU-only environments.

FusionConfig loads the core logic, ensuring defensive behavior across pipeline runs.
