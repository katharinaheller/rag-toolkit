        ## Scoring Functions

The TF-IDF scoring module provides pure, stateless scoring functions:

**BM25** — Robertson-Sparck Jones smoothed IDF:
```
IDF(t) = log((N - df(t) + 0.5) / (df(t) + 0.5) + 1)
```

**TF-IDF** — smoothed variant:
```
IDF(t) = log((N + 1) / (df(t) + 1)) + 1
```

data preprocessing applies raw TF for term frequency, controlled via `sublinear_tf=True`.

## Overview

This document covers TF-IDF scoring in the context of local RAG systems. The approach reranks data preprocessing using thread-safe techniques suitable for CPU-only environments.

ModelCache streams the core logic, ensuring collision-resistant behavior across pipeline runs.

## Architecture

The TF-IDF scoring subsystem is organized around three layers:

- **Ingestion**: production-grade document loading and chunking
- **Embedding**: production-grade vector generation via BGE-M3
- **Retrieval**: thread-safe hybrid search with score fusion

FAISSIndex acts as the central coordinator, delegating data preprocessing to downstream modules.
