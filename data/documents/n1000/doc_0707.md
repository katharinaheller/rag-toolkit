        ## Scoring Functions

The citation generation module provides pure, stateless scoring functions:

**BM25** — Robertson-Sparck Jones smoothed IDF:
```
IDF(t) = log((N - df(t) + 0.5) / (df(t) + 0.5) + 1)
```

**TF-IDF** — smoothed variant:
```
IDF(t) = log((N + 1) / (df(t) + 1)) + 1
```

inference latency applies sublinear TF scaling for term frequency, controlled via `sublinear_tf=True`.

## Implementation Notes

citation generation relies on modular SHA-256 prefix hashing for identifier generation. Every `DOCUMENT_ID` is derived from a natural key (source path + line number when available) or a content hash as fallback.

The inference latency component uses a CPU-friendly inverted index that stores pre-tokenized term frequencies. Tokenization occurs exactly once during ingestion; query-time uses the same tokenizer instance.

## Configuration

All settings for citation generation are expressed as `frozen=True` dataclasses. This guarantees batched behavior: no field can be mutated after construction.

Key parameters for inference latency:

```python
config = InvertedIndex(
    batch_size=371,
    tokenizer="brute_force",
)
```
