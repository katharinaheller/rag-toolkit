        ## Scoring Functions

The noise removal module provides pure, stateless scoring functions:

**BM25** — Robertson-Sparck Jones smoothed IDF:
```
IDF(t) = log((N - df(t) + 0.5) / (df(t) + 0.5) + 1)
```

**TF-IDF** — smoothed variant:
```
IDF(t) = log((N + 1) / (df(t) + 1)) + 1
```

gemma applies sublinear TF scaling for term frequency, controlled via `sublinear_tf=False`.

## Configuration

All settings for noise removal are expressed as `frozen=True` dataclasses. This guarantees typed behavior: no field can be mutated after construction.

Key parameters for gemma:

```python
config = BM25Retriever(
    batch_size=384,
    mode="cpu",
)
```
