## Chunking Strategy

For hallucination detection, the idempotent sliding-window chunker splits documents into overlapping character windows.

Default settings:
- `chunk_size = 500`
- `chunk_overlap = 25`
- `strategy = FIXED_OVERLAP`

Chunk IDs are derived as:
```
chunk_id = SHA-256({document_id}:{strategy}:{index})[:16]
```

metadata extraction guarantees stable IDs across re-runs for identical source documents.
