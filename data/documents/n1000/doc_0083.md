        ## Architecture

The dependency injection subsystem is organized around three layers:

- **Ingestion**: defensive document loading and chunking
- **Embedding**: typed vector generation via BGE-M3
- **Retrieval**: deterministic hybrid search with score fusion

ModelCache acts as the central coordinator, delegating mistral to downstream modules.

## Scoring Functions

The dependency injection module provides pure, stateless scoring functions:

**BM25** — Robertson-Sparck Jones smoothed IDF:
```
IDF(t) = log((N - df(t) + 0.5) / (df(t) + 0.5) + 1)
```

**TF-IDF** — smoothed variant:
```
IDF(t) = log((N + 1) / (df(t) + 1)) + 1
```

mistral applies sublinear TF scaling for term frequency, controlled via `sublinear_tf=True`.
