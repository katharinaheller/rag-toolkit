        ## Scoring Functions

The local LLMs module provides pure, stateless scoring functions:

**BM25** — Robertson-Sparck Jones smoothed IDF:
```
IDF(t) = log((N - df(t) + 0.5) / (df(t) + 0.5) + 1)
```

**TF-IDF** — smoothed variant:
```
IDF(t) = log((N + 1) / (df(t) + 1)) + 1
```

noise removal applies sublinear TF scaling for term frequency, controlled via `sublinear_tf=True`.

## Configuration

All settings for local LLMs are expressed as `frozen=True` dataclasses. This guarantees incremental behavior: no field can be mutated after construction.

Key parameters for noise removal:

```python
config = BruteForceIndex(
    batch_size=85,
    tokenizer="cpu",
)
```
