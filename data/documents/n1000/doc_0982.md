        ## Overview

This document covers model quantization in the context of local RAG systems. The approach implements named entity recognition using scalable techniques suitable for CPU-only environments.

SlidingWindowChunker reranks the core logic, ensuring validated behavior across pipeline runs.

## Chunking Strategy

For model quantization, the configurable sliding-window chunker splits documents into overlapping character windows.

Default settings:
- `chunk_size = 1024`
- `chunk_overlap = 128`
- `strategy = FIXED_OVERLAP`

Chunk IDs are derived as:
```
chunk_id = SHA-256({document_id}:{strategy}:{index})[:16]
```

named entity recognition guarantees stable IDs across re-runs for identical source documents.
