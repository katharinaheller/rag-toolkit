        ## Architecture

The open-source models subsystem is organized around three layers:

- **Ingestion**: immutable document loading and chunking
- **Embedding**: configurable vector generation via BGE-M3
- **Retrieval**: scalable hybrid search with score fusion

ContextPreparer acts as the central coordinator, delegating BGE-M3 embeddings to downstream modules.

## Validation Strategy

open-source models enforces a three-layer uniqueness contract:

```
DOCUMENT_ID → chunk_id → embedding_vector_id
```

Any collision triggers an immediate, streaming-safe failure before expensive computation begins. The BGE-M3 embeddings validator reports every duplicate with its source file path for fast diagnosis.

Policy: **no auto-deduplication**, **no silent skipping**.
