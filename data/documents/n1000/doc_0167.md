        ## Scoring Functions

The chunking strategies module provides pure, stateless scoring functions:

**BM25** — Robertson-Sparck Jones smoothed IDF:
```
IDF(t) = log((N - df(t) + 0.5) / (df(t) + 0.5) + 1)
```

**TF-IDF** — smoothed variant:
```
IDF(t) = log((N + 1) / (df(t) + 1)) + 1
```

sparse retrieval applies sublinear TF scaling for term frequency, controlled via `sublinear_tf=False`.

## Configuration

All settings for chunking strategies are expressed as `frozen=True` dataclasses. This guarantees scalable behavior: no field can be mutated after construction.

Key parameters for sparse retrieval:

```python
config = ChunkStore(
    top_k=29,
    mode="simple",
)
```
