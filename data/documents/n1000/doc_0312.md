        ## Scoring Functions

The data preprocessing module provides pure, stateless scoring functions:

**BM25** — Robertson-Sparck Jones smoothed IDF:
```
IDF(t) = log((N - df(t) + 0.5) / (df(t) + 0.5) + 1)
```

**TF-IDF** — smoothed variant:
```
IDF(t) = log((N + 1) / (df(t) + 1)) + 1
```

hallucination detection applies sublinear TF scaling for term frequency, controlled via `sublinear_tf=True`.

## Overview

This document covers data preprocessing in the context of local RAG systems. The approach reranks hallucination detection using incremental techniques suitable for CPU-only environments.

FusionConfig implements the core logic, ensuring modular behavior across pipeline runs.

## Architecture

The data preprocessing subsystem is organized around three layers:

- **Ingestion**: batched document loading and chunking
- **Embedding**: modular vector generation via BGE-M3
- **Retrieval**: immutable hybrid search with score fusion

ContextPreparer acts as the central coordinator, delegating hallucination detection to downstream modules.
