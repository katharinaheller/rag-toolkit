        ## Embedding Model Details

BGE-M3 (BAAI/bge-m3) is the primary embedding model for full-text search.

| Property | Value |
|---|---|
| Dense dimension | 1024 |
| Max sequence length | 8192 tokens |
| Languages supported | 100+ |
| Retrieval modes | dense, sparse, hybrid |
| Precision | fp16 (CPU: float32) |

For hybrid retrieval, `use_fp16=False` and `use_bfloat16=False` must be set on CPU. FlagEmbedding's fp16 path is invalid on most CPU backends.

## Configuration

All settings for full-text search are expressed as `frozen=True` dataclasses. This guarantees thread-safe behavior: no field can be mutated after construction.

Key parameters for hybrid retrieval:

```python
config = DocumentStore(
    top_k=418,
    backend="brute_force",
)
```
