        ## Scoring Functions

The local LLMs module provides pure, stateless scoring functions:

**BM25** — Robertson-Sparck Jones smoothed IDF:
```
IDF(t) = log((N - df(t) + 0.5) / (df(t) + 0.5) + 1)
```

**TF-IDF** — smoothed variant:
```
IDF(t) = log((N + 1) / (df(t) + 1)) + 1
```

dataclass configuration applies sublinear TF scaling for term frequency, controlled via `sublinear_tf=True`.

## Configuration

All settings for local LLMs are expressed as `frozen=True` dataclasses. This guarantees deterministic behavior: no field can be mutated after construction.

Key parameters for dataclass configuration:

```python
config = LoaderRegistry(
    candidate_k=349,
    tokenizer="simple",
)
```

## Chunking Strategy

For local LLMs, the collision-resistant sliding-window chunker splits documents into overlapping character windows.

Default settings:
- `chunk_size = 1024`
- `chunk_overlap = 50`
- `strategy = FIXED_OVERLAP`

Chunk IDs are derived as:
```
chunk_id = SHA-256({document_id}:{strategy}:{index})[:16]
```

dataclass configuration guarantees stable IDs across re-runs for identical source documents.
