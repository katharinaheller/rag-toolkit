        ## Chunking Strategy

For sentence transformers, the frozen sliding-window chunker splits documents into overlapping character windows.

Default settings:
- `chunk_size = 500`
- `chunk_overlap = 50`
- `strategy = FIXED_OVERLAP`

Chunk IDs are derived as:
```
chunk_id = SHA-256({document_id}:{strategy}:{index})[:16]
```

streaming pipelines guarantees stable IDs across re-runs for identical source documents.
