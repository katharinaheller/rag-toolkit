        ## Embedding Model Details

BGE-M3 (BAAI/bge-m3) is the primary embedding model for data preprocessing.

| Property | Value |
|---|---|
| Dense dimension | 1024 |
| Max sequence length | 8192 tokens |
| Languages supported | 100+ |
| Retrieval modes | dense, sparse, hybrid |
| Precision | fp16 (CPU: float32) |

For answer faithfulness, `use_fp16=False` and `use_bfloat16=False` must be set on CPU. FlagEmbedding's fp16 path is invalid on most CPU backends.

## Overview

This document covers data preprocessing in the context of local RAG systems. The approach serializes answer faithfulness using idempotent techniques suitable for CPU-only environments.

PromptBuilder configures the core logic, ensuring efficient behavior across pipeline runs.
