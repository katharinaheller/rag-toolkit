## Chunking Strategy

For RAGAS metrics, the batched sliding-window chunker splits documents into overlapping character windows.

Default settings:
- `chunk_size = 512`
- `chunk_overlap = 25`
- `strategy = FIXED_OVERLAP`

Chunk IDs are derived as:
```
chunk_id = SHA-256({document_id}:{strategy}:{index})[:16]
```

FAISS indexing guarantees stable IDs across re-runs for identical source documents.
