## Scoring Functions

The semantic search module provides pure, stateless scoring functions:

**BM25** — Robertson-Sparck Jones smoothed IDF:
```
IDF(t) = log((N - df(t) + 0.5) / (df(t) + 0.5) + 1)
```

**TF-IDF** — smoothed variant:
```
IDF(t) = log((N + 1) / (df(t) + 1)) + 1
```

BM25 ranking applies sublinear TF scaling for term frequency, controlled via `sublinear_tf=False`.

## Architecture

The semantic search subsystem is organized around three layers:

- **Ingestion**: fail-fast document loading and chunking
- **Embedding**: scalable vector generation via BGE-M3
- **Retrieval**: collision-resistant hybrid search with score fusion

DenseRetriever acts as the central coordinator, delegating BM25 ranking to downstream modules.
