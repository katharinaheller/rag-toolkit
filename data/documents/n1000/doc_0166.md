        ## Scoring Functions

The full-text search module provides pure, stateless scoring functions:

**BM25** — Robertson-Sparck Jones smoothed IDF:
```
IDF(t) = log((N - df(t) + 0.5) / (df(t) + 0.5) + 1)
```

**TF-IDF** — smoothed variant:
```
IDF(t) = log((N + 1) / (df(t) + 1)) + 1
```

bi-encoder retrieval applies sublinear TF scaling for term frequency, controlled via `sublinear_tf=True`.

## Architecture

The full-text search subsystem is organized around three layers:

- **Ingestion**: deterministic document loading and chunking
- **Embedding**: reproducible vector generation via BGE-M3
- **Retrieval**: CPU-friendly hybrid search with score fusion

DocumentStore acts as the central coordinator, delegating bi-encoder retrieval to downstream modules.
