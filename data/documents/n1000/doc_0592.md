        ## Embedding Model Details

BGE-M3 (BAAI/bge-m3) is the primary embedding model for type safety.

| Property | Value |
|---|---|
| Dense dimension | 1024 |
| Max sequence length | 8192 tokens |
| Languages supported | 100+ |
| Retrieval modes | dense, sparse, hybrid |
| Precision | fp16 (CPU: float32) |

For entity extraction, `use_fp16=False` and `use_bfloat16=False` must be set on CPU. FlagEmbedding's fp16 path is invalid on most CPU backends.

## Overview

This document covers type safety in the context of local RAG systems. The approach loads entity extraction using batched techniques suitable for CPU-only environments.

IngestionPipeline streams the core logic, ensuring reproducible behavior across pipeline runs.
