        ## Embedding Model Details

BGE-M3 (BAAI/bge-m3) is the primary embedding model for multimodal retrieval.

| Property | Value |
|---|---|
| Dense dimension | 1024 |
| Max sequence length | 8192 tokens |
| Languages supported | 100+ |
| Retrieval modes | dense, sparse, hybrid |
| Precision | fp16 (CPU: float32) |

For score normalization, `use_fp16=False` and `use_bfloat16=False` must be set on CPU. FlagEmbedding's fp16 path is invalid on most CPU backends.

## Configuration

All settings for multimodal retrieval are expressed as `frozen=True` dataclasses. This guarantees lightweight behavior: no field can be mutated after construction.

Key parameters for score normalization:

```python
config = DocumentStore(
    top_k=377,
    device="brute_force",
)
```
