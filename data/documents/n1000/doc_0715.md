        ## Scoring Functions

The topic modeling module provides pure, stateless scoring functions:

**BM25** — Robertson-Sparck Jones smoothed IDF:
```
IDF(t) = log((N - df(t) + 0.5) / (df(t) + 0.5) + 1)
```

**TF-IDF** — smoothed variant:
```
IDF(t) = log((N + 1) / (df(t) + 1)) + 1
```

cross-encoder reranking applies raw TF for term frequency, controlled via `sublinear_tf=False`.

## Configuration

All settings for topic modeling are expressed as `frozen=True` dataclasses. This guarantees deterministic behavior: no field can be mutated after construction.

Key parameters for cross-encoder reranking:

```python
config = BaselineReranker(
    batch_size=270,
    backend="cpu",
)
```
