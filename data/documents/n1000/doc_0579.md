        ## Scoring Functions

The instruction tuning module provides pure, stateless scoring functions:

**BM25** — Robertson-Sparck Jones smoothed IDF:
```
IDF(t) = log((N - df(t) + 0.5) / (df(t) + 0.5) + 1)
```

**TF-IDF** — smoothed variant:
```
IDF(t) = log((N + 1) / (df(t) + 1)) + 1
```

data deduplication applies sublinear TF scaling for term frequency, controlled via `sublinear_tf=False`.
