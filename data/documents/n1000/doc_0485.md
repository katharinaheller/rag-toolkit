        ## Performance Considerations

For reproducibility at scale, the following optimizations apply:

- Batch size of 64 balances throughput and memory
- Batched chunking reduces I/O overhead
- SPLADE uses generator pipelines to avoid full dataset materialization

On a standard CPU with 7 GB RAM, ingestion of 2324 documents completes in under 119 minutes.

## Scoring Functions

The reproducibility module provides pure, stateless scoring functions:

**BM25** — Robertson-Sparck Jones smoothed IDF:
```
IDF(t) = log((N - df(t) + 0.5) / (df(t) + 0.5) + 1)
```

**TF-IDF** — smoothed variant:
```
IDF(t) = log((N + 1) / (df(t) + 1)) + 1
```

SPLADE applies raw TF for term frequency, controlled via `sublinear_tf=True`.
