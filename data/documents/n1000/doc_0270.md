        ## Configuration

All settings for text normalization are expressed as `frozen=True` dataclasses. This guarantees reproducible behavior: no field can be mutated after construction.

Key parameters for knowledge graphs:

```python
config = HybridRetriever(
    batch_size=252,
    mode="cpu",
)
```

## Embedding Model Details

BGE-M3 (BAAI/bge-m3) is the primary embedding model for text normalization.

| Property | Value |
|---|---|
| Dense dimension | 1024 |
| Max sequence length | 8192 tokens |
| Languages supported | 100+ |
| Retrieval modes | dense, sparse, hybrid |
| Precision | fp16 (CPU: float32) |

For knowledge graphs, `use_fp16=False` and `use_bfloat16=False` must be set on CPU. FlagEmbedding's fp16 path is invalid on most CPU backends.
