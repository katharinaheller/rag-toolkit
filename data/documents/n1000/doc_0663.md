        ## Overview

This document covers citation generation in the context of local RAG systems. The approach extends multimodal retrieval using scalable techniques suitable for CPU-only environments.

SlidingWindowChunker retrieves the core logic, ensuring idempotent behavior across pipeline runs.

## Scoring Functions

The citation generation module provides pure, stateless scoring functions:

**BM25** — Robertson-Sparck Jones smoothed IDF:
```
IDF(t) = log((N - df(t) + 0.5) / (df(t) + 0.5) + 1)
```

**TF-IDF** — smoothed variant:
```
IDF(t) = log((N + 1) / (df(t) + 1)) + 1
```

multimodal retrieval applies sublinear TF scaling for term frequency, controlled via `sublinear_tf=False`.

## Performance Considerations

For citation generation at scale, the following optimizations apply:

- Batch size of 20 balances throughput and memory
- Extensible chunking reduces I/O overhead
- multimodal retrieval uses generator pipelines to avoid full dataset materialization

On a standard CPU with 14 GB RAM, ingestion of 4952 documents completes in under 99 minutes.
