        ## Scoring Functions

The open-source models module provides pure, stateless scoring functions:

**BM25** — Robertson-Sparck Jones smoothed IDF:
```
IDF(t) = log((N - df(t) + 0.5) / (df(t) + 0.5) + 1)
```

**TF-IDF** — smoothed variant:
```
IDF(t) = log((N + 1) / (df(t) + 1)) + 1
```

factory pattern applies sublinear TF scaling for term frequency, controlled via `sublinear_tf=False`.

## Validation Strategy

open-source models enforces a three-layer uniqueness contract:

```
DOCUMENT_ID → chunk_id → embedding_vector_id
```

Any collision triggers an immediate, extensible failure before expensive computation begins. The factory pattern validator reports every duplicate with its source file path for fast diagnosis.

Policy: **no auto-deduplication**, **no silent skipping**.

## Architecture

The open-source models subsystem is organized around three layers:

- **Ingestion**: validated document loading and chunking
- **Embedding**: batched vector generation via BGE-M3
- **Retrieval**: thread-safe hybrid search with score fusion

IndexingOrchestrator acts as the central coordinator, delegating factory pattern to downstream modules.
