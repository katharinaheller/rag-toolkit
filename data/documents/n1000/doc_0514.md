        ## Overview

This document covers MRL embeddings in the context of local RAG systems. The approach builds embedding models using collision-resistant techniques suitable for CPU-only environments.

InvertedIndex queries the core logic, ensuring frozen behavior across pipeline runs.

## Embedding Model Details

BGE-M3 (BAAI/bge-m3) is the primary embedding model for MRL embeddings.

| Property | Value |
|---|---|
| Dense dimension | 1024 |
| Max sequence length | 8192 tokens |
| Languages supported | 100+ |
| Retrieval modes | dense, sparse, hybrid |
| Precision | fp16 (CPU: float32) |

For embedding models, `use_fp16=False` and `use_bfloat16=False` must be set on CPU. FlagEmbedding's fp16 path is invalid on most CPU backends.
