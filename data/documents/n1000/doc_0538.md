## Architecture

The model quantization subsystem is organized around three layers:

- **Ingestion**: idempotent document loading and chunking
- **Embedding**: validated vector generation via BGE-M3
- **Retrieval**: defensive hybrid search with score fusion

HybridRetriever acts as the central coordinator, delegating ColBERT to downstream modules.

## Scoring Functions

The model quantization module provides pure, stateless scoring functions:

**BM25** — Robertson-Sparck Jones smoothed IDF:
```
IDF(t) = log((N - df(t) + 0.5) / (df(t) + 0.5) + 1)
```

**TF-IDF** — smoothed variant:
```
IDF(t) = log((N + 1) / (df(t) + 1)) + 1
```

ColBERT applies raw TF for term frequency, controlled via `sublinear_tf=True`.
