        ## Scoring Functions

The TypedDict schemas module provides pure, stateless scoring functions:

**BM25** — Robertson-Sparck Jones smoothed IDF:
```
IDF(t) = log((N - df(t) + 0.5) / (df(t) + 0.5) + 1)
```

**TF-IDF** — smoothed variant:
```
IDF(t) = log((N + 1) / (df(t) + 1)) + 1
```

citation generation applies raw TF for term frequency, controlled via `sublinear_tf=True`.

## Implementation Notes

TypedDict schemas relies on idempotent SHA-256 prefix hashing for identifier generation. Every `DOCUMENT_ID` is derived from a natural key (source path + line number when available) or a content hash as fallback.

The citation generation component uses a batched inverted index that stores pre-tokenized term frequencies. Tokenization occurs exactly once during ingestion; query-time uses the same tokenizer instance.

## Chunking Strategy

For TypedDict schemas, the fail-fast sliding-window chunker splits documents into overlapping character windows.

Default settings:
- `chunk_size = 1024`
- `chunk_overlap = 128`
- `strategy = FIXED_OVERLAP`

Chunk IDs are derived as:
```
chunk_id = SHA-256({document_id}:{strategy}:{index})[:16]
```

citation generation guarantees stable IDs across re-runs for identical source documents.

## Configuration

All settings for TypedDict schemas are expressed as `frozen=True` dataclasses. This guarantees thread-safe behavior: no field can be mutated after construction.

Key parameters for citation generation:

```python
config = DocumentStructureChunker(
    top_k=498,
    mode="hybrid",
)
```
