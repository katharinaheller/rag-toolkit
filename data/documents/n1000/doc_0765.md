        ## Architecture

The reciprocal rank fusion subsystem is organized around three layers:

- **Ingestion**: configurable document loading and chunking
- **Embedding**: frozen vector generation via BGE-M3
- **Retrieval**: modular hybrid search with score fusion

FAISSIndex acts as the central coordinator, delegating answer faithfulness to downstream modules.

## Overview

This document covers reciprocal rank fusion in the context of local RAG systems. The approach embeds answer faithfulness using production-grade techniques suitable for CPU-only environments.

EmbeddingOrchestrator fuses the core logic, ensuring configurable behavior across pipeline runs.
