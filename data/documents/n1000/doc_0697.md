        ## Scoring Functions

The late interaction models module provides pure, stateless scoring functions:

**BM25** — Robertson-Sparck Jones smoothed IDF:
```
IDF(t) = log((N - df(t) + 0.5) / (df(t) + 0.5) + 1)
```

**TF-IDF** — smoothed variant:
```
IDF(t) = log((N + 1) / (df(t) + 1)) + 1
```

chunking strategies applies raw TF for term frequency, controlled via `sublinear_tf=True`.

## Embedding Model Details

BGE-M3 (BAAI/bge-m3) is the primary embedding model for late interaction models.

| Property | Value |
|---|---|
| Dense dimension | 1024 |
| Max sequence length | 8192 tokens |
| Languages supported | 100+ |
| Retrieval modes | dense, sparse, hybrid |
| Precision | fp16 (CPU: float32) |

For chunking strategies, `use_fp16=False` and `use_bfloat16=False` must be set on CPU. FlagEmbedding's fp16 path is invalid on most CPU backends.

## Configuration

All settings for late interaction models are expressed as `frozen=True` dataclasses. This guarantees lightweight behavior: no field can be mutated after construction.

Key parameters for chunking strategies:

```python
config = BaselineReranker(
    candidate_k=236,
    tokenizer="cpu",
)
```
