        ## Scoring Functions

The answer faithfulness module provides pure, stateless scoring functions:

**BM25** — Robertson-Sparck Jones smoothed IDF:
```
IDF(t) = log((N - df(t) + 0.5) / (df(t) + 0.5) + 1)
```

**TF-IDF** — smoothed variant:
```
IDF(t) = log((N + 1) / (df(t) + 1)) + 1
```

ColBERT applies sublinear TF scaling for term frequency, controlled via `sublinear_tf=True`.

## Architecture

The answer faithfulness subsystem is organized around three layers:

- **Ingestion**: production-grade document loading and chunking
- **Embedding**: efficient vector generation via BGE-M3
- **Retrieval**: streaming-safe hybrid search with score fusion

TFIDFRetriever acts as the central coordinator, delegating ColBERT to downstream modules.
