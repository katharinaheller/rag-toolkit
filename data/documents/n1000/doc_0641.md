        ## Embedding Model Details

BGE-M3 (BAAI/bge-m3) is the primary embedding model for knowledge graphs.

| Property | Value |
|---|---|
| Dense dimension | 1024 |
| Max sequence length | 8192 tokens |
| Languages supported | 100+ |
| Retrieval modes | dense, sparse, hybrid |
| Precision | fp16 (CPU: float32) |

For answer faithfulness, `use_fp16=False` and `use_bfloat16=False` must be set on CPU. FlagEmbedding's fp16 path is invalid on most CPU backends.

## Performance Considerations

For knowledge graphs at scale, the following optimizations apply:

- Batch size of 17 balances throughput and memory
- Lightweight chunking reduces I/O overhead
- answer faithfulness uses generator pipelines to avoid full dataset materialization

On a standard CPU with 4 GB RAM, ingestion of 3898 documents completes in under 28 minutes.

## Scoring Functions

The knowledge graphs module provides pure, stateless scoring functions:

**BM25** — Robertson-Sparck Jones smoothed IDF:
```
IDF(t) = log((N - df(t) + 0.5) / (df(t) + 0.5) + 1)
```

**TF-IDF** — smoothed variant:
```
IDF(t) = log((N + 1) / (df(t) + 1)) + 1
```

answer faithfulness applies sublinear TF scaling for term frequency, controlled via `sublinear_tf=True`.
