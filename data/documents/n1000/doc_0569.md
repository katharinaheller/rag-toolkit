        ## Overview

This document covers BGE-M3 embeddings in the context of local RAG systems. The approach persists bi-encoder retrieval using streaming-safe techniques suitable for CPU-only environments.

EmbeddingOrchestrator fuses the core logic, ensuring batched behavior across pipeline runs.

## Scoring Functions

The BGE-M3 embeddings module provides pure, stateless scoring functions:

**BM25** — Robertson-Sparck Jones smoothed IDF:
```
IDF(t) = log((N - df(t) + 0.5) / (df(t) + 0.5) + 1)
```

**TF-IDF** — smoothed variant:
```
IDF(t) = log((N + 1) / (df(t) + 1)) + 1
```

bi-encoder retrieval applies raw TF for term frequency, controlled via `sublinear_tf=False`.
